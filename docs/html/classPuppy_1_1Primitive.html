Detailed Description

Genetic programming abstract primitive class.

A primitive is an abstract class of terminals and functions that can be used in a GP program. Concrete primitives must define method execute, which define the characteristic operation to execute. Primitives are generally heap allocated (with a call to the new operator) and managed by the associated smart pointer defined in class PrimitiveHandle.
